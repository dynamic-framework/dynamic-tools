Test Dynamic Theme Color Changes

Instructions:

  1. Open the main application at index.html
  2. Click on "Edit Mode" button in the CSS Code tab
  3. Change --bs-primary: #d81b60; to --bs-primary: green;
  4. Click "Apply Changes"
  5. Verify that primary color elements change to green

Test Color Formats

Test these different color formats in the CSS editor:

  • green - Named color
  • #00FF00 - Hex color
  • rgb(0, 255, 0) - RGB format
  • rgba(0, 255, 0, 1) - RGBA format
  • hsl(120, 100%, 50%) - HSL format
